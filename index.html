Linguagem de Marcação

Uma Linguagem de Marcação é um conjunto de sinais e códigos aplicados a um texto ou a dados para definir formatos, maneiras de exibição e padrões.

Linguagem de Marcação utilizam mercadores ou tags, que quando são identificados em um sistema, trazem definições de como um determinado conteúdo deve ser exibido.

Existem três tipos de classificação para linguagens de marcação:

  • Marcação de apresentação: marcações incorporadas no texto para apresentar linguagens renderizadas, costumam ser utilizadas por processadores de texto.
  • Marcação procedimental: fornece instruções para o processamento do texto por programas, alguns exemplos são troff, TeX e PostScript.
  • Marcação descritiva: oferece rotulações de partes do documento com o instuito de dissociar a estrutura do documento e seu processamento, alguns exemplos são o HTML e XML.

A primeira linguagem de marcação criada foi a SGML (Standard Generalized Markup Language) no final da década de 60 com o objetivo de construir um sistema portável para o intercâmbio e manipulação de documentos.

O desenvolvimento da SGML possibilitou o surgimento do HTML (HyperText Markup Language) no final da década de 80, e revolucionou a maneira de visualização das páginas na Web.

O HTML foi criado por Tim Berners-Lee, uma linguagem que permitia acrescentar elementos nos textos usados por ele e um grupo de pesquisadores, na publicação dos seus trabalhos no inicio da Internet.

O HTML foi atualizado em cinco versões até chegar a 5HTML, em 2014.

Em 1996, o engenheiro Jon Bosak desenvolveu o XML (eXtensible Markup Language) também com base na linguagem SGML, com o intuito de utilizar as marcações para descrever a estrutura dos conteúdos.

Em 2000, foi desenvolvida a XHTML (eXtensible Hypertext Markup Language), combinando entre si as linguagens HTML e XML. Ela foi responsável pela melhoria na exibição da Web em outros dispositivos além do computador, como os smartphones e na acessiblidade do conteúdo.

Tim Benners-Lee

Página pessoal do Sir Timonthy Berners-Lee.

Vamos trazer alguns pontos de interesse da jornada do Tim Benners-Lee.

  • Em 1989, enquanto trabalhava no CERN em Genebra, Suíça, Tim Berners-Lee propôs o projeto do World Wide Web. Baseado em trabalhos anteriores, o objetivo era permitir a colaboração ao conectar documentos de hipertexto. Berners-Lee desenvolveu o primeiro servidor web, httpd, e o primeiro cliente, WorldWideWeb, um navegador/editor de hipertexto para o ambiente NeXTStep. O desenvolvimento começou em outubro de 1990, com o programa WorldWideWeb disponível inicialmente dentro do CERN em dezembro de 1990 e na Internet em geral no verão de 1991.
  • Durante os anos de 1991 a 1993, Tim Berners-Lee persistiu no aprimoramento do design da Web, colaborando com usuários ao redor da Internet. Suas primeiras especificações de URIs, HTTP e HTML foram refinadas e debatidas em diversos círculos conforme a tecnologia web se expandia.
  • Tim Berners-Lee se formou no The Queen's College na Oxford University, Inglaterra, em 1976. Durante esse período, ele construiu seu primeiro computador usando um ferro de solda, portas TTL, um processador M6800 e uma televisão antiga.
  • Durante dois anos, ele trabalhou na Plessey Telecommunications Ltd, no Reino Unido, focando em sistemas de transações distribuídas, relés de mensagens e tecnologia de código de barras.
  • Em 1978, Tim deixou a Plessey para ingressar na D.G Nash Ltd, onde desenvolveu, entre outras coisas, software de composição para impressoras inteligentes e um sistema operacional multitarefa.
  • Após um ano e meio como consultor independente, Tim passou seis meses (junho a dezembro de 1980) como engenheiro de software consultor no CERN. Durante esse período, ele desenvolveu seu primeiro programa para armazenar informações, incluindo o uso de associações aleatórias, chamado Enquire. Embora nunca tenha sido publicado, esse programa serviu como base conceitual para o desenvolvimento futuro da World Wide Web.
  • De 1981 a 1984, Tim trabalhou na Image Computer Systems Ltd, onde foi responsável pelo design técnico. Seu trabalho envolveu desenvolvimento de firmware para controle em tempo real, software gráfico e de comunicação, além de uma linguagem macro genérica. Em 1984, ele ingressou no CERN como bolsista, focando em sistemas distribuídos em tempo real para aquisição de dados científicos e controle de sistema, contribuindo para o desenvolvimento do software do sistema FASTBUS e projetando um sistema de chamada de procedimento remoto heterogêneo.
  • Em 1994, Tim Berners-Lee fundou o Consórcio World Wide Web no Laboratório de Ciência da Computação (LCS), que posteriormente se tornou parte do Laboratório de Ciência da Computação e Inteligência Artificial (CSAIL) do Massachusetts Institute of Technology (MIT). Ele é Diretor Emérito do W3C, uma organização que desenvolve tecnologias para a Web.
  • Em 1999, Tim Berners-Lee tornou-se o primeiro titular da Cadeira 3Com Founders no MIT. Em 2008, foi nomeado Professor de Engenharia 3COM Founders na Escola de Engenharia, com um cargo conjunto no Departamento de Engenharia Elétrica e Ciência da Computação no CSAIL, onde agora é Professor Emérito. Ele também liderou o Grupo de Informações Descentralizadas (DIG), trabalhando em projetos para descentralizar a Web, garantir os direitos dos usuários em big data e analytics, e utilizar tecnologias móveis em situações de desastre e na sociedade.
  • O Grupo de Informações Descentralizadas lidera o Solid Project para devolver às pessoas o controle de seus dados e descentralizar a Web. Tim Berners-Lee é co-fundador e CTO da inrupt, uma empresa que apoia a plataforma Solid e sua comunidade de código aberto.
  • Em 2016, Sir Tim Berners-Lee ingressou no Computer Science Department at the University of Oxford como professor.
  • Em 2009, Tim Berners-Lee fundou e tornou-se diretor da World Wide Web Foundation. Esta é uma organização sem fins lucrativos dedicada a alcançar um mundo onde todas as pessoas possam usar a Web para se comunicar, colaborar e inovar livremente. A fundação trabalha para financiar e coordenar esforços para defender a Web Aberta e promover seu potencial para beneficiar a humanidade.
  • Em junho de 2009, o Primeiro-Ministro Gordon Brown anunciou a colaboração com Tim Berners-Lee para tornar os dados mais acessíveis na web, impulsionando a transparência do governo no Reino Unido. Berners-Lee foi membro do Public Sector Transparency Board e promoveu a abertura de dados governamentais globalmente.
  • Em 2011, Tim Berners-Lee foi nomeado para o Conselho de Administração da Ford Foundation, enquanto preside o Open Data Institute do Reino Unido, uma iniciativa que ele co-fundou em 2012 com Sir Nigel Shadbolt. O instituto visa promover o uso de dados abertos para gerar valor econômico, ambiental e social.
  • Tim Berners-Lee é co-autor, junto com Mark Fischetti, do livro Weaving the Web, publicado em 1999. O livro explora o passado, presente e futuro da Web.
  • Em 18 de março de 2013, Sir Tim Berners-Lee, juntamente com Vinton Cerf, Robert Kahn, Louis Pouzin e Marc Andreesen, foi agraciado com o Prêmio Queen Elizabeth Prize for Engineering por "inovações revolucionárias em engenharia que beneficiaram globalmente a humanidade".
  • Em 4 de abril de 2017, Sir Tim Berners-Lee foi premiado com o Prêmio ACM A.M. Turing Award por inventar a World Wide Web, o primeiro navegador web e os protocolos e algoritmos fundamentais que permitiram a escalabilidade da Web. O Prêmio Turing, conhecido como o "Prêmio Nobel da Computação", é considerado uma das mais prestigiadas distinções na área da Ciência da Computação.

Tags de Título

Os títulos vão do seu maior nível hierárquico que é <h1> até a de menor valor hierárquico <h6>.

Atributos

Os atributos são usados para personalizar as tags, modificando sua estrutura ou funcionalidade.

A maioria das tags em seus proṕrios atributos. Contudo, existem alguns atributos genéricos que podem ser utilizados na maioria das tags HTML.

Tags Descrição
class="..." Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos)
id="..." Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento)
style="..." Permite incluir elementos CSS (estilos) dentro da tag
lang="..." Define o idioma principal do elemento
title="..." Define o título do elemento
alt="..." Define um texto alternativo e, por isso, é muito utilizado em imagens, auxilia nas práticas de SEO
hidden Oculta o elemento
align="..." Permite definir o padrão de alinhamento desse elemento, como por exemplo: right, center, left e justify;
width="..." Define uma largura para o elemento
heigth="..." Define uma altura para o elemento

<input /> É utilizada para entrada de dados, porém deve ser acompanhada pelo type

A tag input possui vários atributos que server para auxiliar o input, um dos atributos que é bastante utilizado é o placeholder="..."que escreve dentro do campo input, uma mensagem que for definida.

É utilizado o type="..." para definir o tipo de entrada que vai ser inserido naquele campo, auxiliando na validação de dados, pois se for definido que o type="email" o que é esperado de entrada nesse campo é um email, já possuindo algumas validações do próprio HTML.
